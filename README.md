Task Management API — Documentation Outline
1. Project Overview

A backend API built using Node.js, Express, MongoDB (Mongoose ORM), and JWT authentication.
The API allows users to sign up, log in and manage their personal tasks with full CRUD operations, filtering, searching, and pagination.

2. Tech Stack

Node.js + Express — Backend framework
MongoDB + Mongoose — Database and ODM
JWT (jsonwebtoken) — Authentication
bcrypt — Password hashing
Zod (optional) — Validation
Postman — API testing
